## Úkol
Postav profesionální demo web pro tuto firmu:

1. **index.html** je šablona — přepiš VEŠKERÝ placeholder obsah podle dat výše
2. V **img/** jsou stažené fotky z webu firmy — **POUŽIJ JE** (hero background, galerie, reference sekce)
3. Vytvoř **podstránky** (sluzby.html, kontakt.html, případně další dle šablony)
4. Kontaktní formulář: `data-netlify="true" name="demo-kontakt"`
5. Formulář musí mít honeypot: `<input type="hidden" name="bot-field">`
6. Obrázky referencuj jako `img/photo-1.jpg`, `img/photo-2.jpg` atd.
7. Texty piš **česky**, **vykání**, `&nbsp;` před jednopísmennými předložkami (k, s, v, z, o, u, i, a)
8. Po dokončení: `git add -A && git commit -m "Demo web" && git push`
